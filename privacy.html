Privacy Policy – AimForge

Last updated: 2026

1. Overview

AimForge is a public analytics platform for VALORANT players. We use official Riot Games APIs to retrieve authorized player data.

2. Data We Collect

  • Riot PUUID (when user signs in via Riot Sign-On)
  • Match statistics (when permitted by Riot API)
  • Calculated performance metrics (GSI, improvement indicators)
  • Timestamps for analysis tracking

3. How Data Is Used

Data is used solely for performance analysis, ranking, and personalized coaching insights.

4. Data Storage

We store minimal necessary data. Users may request deletion of their data at any time.

5. No Game Interaction

AimForge does not interact with the VALORANT game client, does not read memory, does not bypass Vanguard, and does not provide any competitive advantage.

AimForge is not endorsed by Riot Games. VALORANT and Riot Games are trademarks of Riot Games, Inc.
